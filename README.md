# node-api-boilerplate
Just a boiler plate for new NODE APIs using the MVC pattern

## Setup

```bash
yarn add
# or npm install --save
```

## Run (dev)

```bash
node start
```

## Run (build)

```bash
# first, build the project
yarn run build

# Run with pm2
pm2 start --name app-name -i max dist/index.js
```
